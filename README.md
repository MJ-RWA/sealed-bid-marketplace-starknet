# Private Sealed-Bid Marketplace (Starknet)

A privacy-preserving marketplace where Web3 service providers submit sealed bids for jobs.

## MVP Features
- Create job
- Submit sealed bid
- Reveal bid
- On-chain winner selection

## Tech Stack
- Starknet (Cairo)
- React
- Node.js
- Notion (task tracking)

## Team Structure
- contracts/ → Cairo Dev
- frontend/ → Frontend Dev
- backend/ → Backend Dev
